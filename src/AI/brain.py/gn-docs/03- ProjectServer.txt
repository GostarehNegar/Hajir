پراجکت سرور یکی از نرم افزارهای سامانه مدیریت پروژه مایکروسافت (EPM) بوده و از آن جهت مدیریت زمان‌بندی، منابع انسانی، هزینه، تقاضا، یکپارچگی اطلاعات، یکپارچگی تیم پروژه، گزارش‌گیری‌ از پروژه، کنترل محدوده پروژه و ... استفاده می‌شود. برخلاف نرم افزار مایکروسافت پراجکت، در Microsoft Project Server به طور همزمان می‌توان چندین پروژه را کنترل کرد لذا پراجکت سرور برای کنترل به هر یک از پروژه ها یک سایت شیرپونتی اختصاص داده و در در آن سایت مستندات، گزارش‌های کنترل پروژه، موارد قابل تحویل، محصولات پروژه و وقایع مربوط به پروژه قرار دارد و افراد می‌توانند با نام کاربری و پسورد خود وارد شده و مطابق دسترسی تعریف شده وظایف خود را کامل و یا تغییر دهند. یکی از نکات قابل توجه در پراجکت سرور قابل ویرایش بودن برنامه زمانبندی تحت وب است.

در واقع پراجکت سرور همان مایکروسافت پراجکت تحت وب است که امکانات بیشتری را درحوزه کنترل پروژه برای کاربران فراهم کرده و برای کارشناسان کنترل پروژه امکان مدیریت همزمان چند پروژه را فراهم می کند.

محیط برنامه Microsoft Project Server شبیه Microsoft Project بوده و می توان اطلاعات هر پروژه را به طور جداگانه در آن وارد کرد. هر پروژه دارای فعالیت های مختلف، مدت زمان مربوط به هر فعالیت، منابع اختصاص داده شده، برنامه زمانبندی، تاریخ شروع و پایان، مدیر پروژه، ناظر و ... می باشد. این اطلاعات در دیتابیس Project Server وارد شده و در فضای کاری آن قابل مشاهده می باشند.

همچنین بانک اطلاعاتی Project Server كاربرگ ها، گزارشات وضعیت، گزارشات و مدل‌های سبد پروژه‌ها، منابع سازمانی، الگوی جامع سازمانی، و الگو‌ها را برای تعریف سریعتر پروژه‌های جدید فراهم می کند. در پراجکت سرور فیلدهای اطلاعاتی مختلف و متعددی قابل تعریف بوده که با استفاده از آنها اطلاعات جامعی از پروژه وارد سامانه مدیریت پروژه می شود.

پراجکت سرور در صفحه‌ای مجزا می‌تواند تمامی پروژه های سازمان را همراه با فیلدهای اطلاعاتی و فیلتر های مورد نظر نشان داده و مدیران پروژه و ارشد را در جریان آخرین اطلاعات پروژه قرار دهد.

ویژگی های نرم افزار Project Server

از پراجکت سرور در حوزه‌های مختلفی استفاده می‌شود از جمله:

    مدیریت زمانبندی
    مدیریت منابع
    مدیریت سبد پروژه ها
    مدیریت وظایف
    مدیریت تقاضا
    مدیریت هزینه
    هوش سازمانی
    یکپارچگی تیم پروژه

فارسی ساز Microsoft Project Server پرنیان محصول گستره نگار قادر است تا محیط سایت پروژه را کاملا فارسی و تقویم شمسی را در آن اعمال نماید.
