قفل سخت افزاری
برای استفاده از پرنیان لازم است قفل سخت افزاری داشته باشید. این قفل سخت افزاری روی پورت USB وصل میشود. شما میتوانید از قفل تحت شبکه هم استفاده کنید. معمولا این قفل روی یک سرور نصب میشود و استفاده کنندگان در شبکه میتوانند به این سرور متصل شوند. قفلهای تحت شبکه به تعداد معینی از کاربرها اجازه استفاده میدهند و تعداد این کاربران به هنگام فروش روی این قفلها تنظیم میشود. از همین رو است که قیمت نسخه های تحت شبکه بر اساس تعداد کاربران متفاوت است.

برای نرم‌افزارهای تحت وب نظیر داینامیکس، سی.آر.ام و شیرپوینت تنها یک قفل روی سرور نصب میشود. از آنجاییکه ممکن است نصب یک قفل سخت افزاری روی سرور بعضا مشکل باشد، برای نرم افزارهای سروری قفل نرم افزاری هم قابل استفاده است.

قفل نرم افزاری
قفل نرم افزاری در واقع یک لایسنس فایل است که برای یک کامپیوتر خاص ساخته میشود. ابتدا درخواستی به شکل یک پرونده ی xml روی کامپیوتر مقصد ساخته میشود. این درخواست برای شرکت فرستاده میشود. شرکت آنرا تایید و امضا میکند. نسخه ی امضا شده روی کامپیوتر مقصد قرار داده میشود.
